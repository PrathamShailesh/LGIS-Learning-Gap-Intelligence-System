# Learning Gap Intelligence System (LGIS)

## Overview

The Learning Gap Intelligence System (LGIS) is a data-driven analytics and decision-support platform built using ASER educational datasets. The project helps education administrators identify learning gaps, monitor performance trends, and make data-informed intervention decisions.

The system transforms static educational reports into actionable insights through interactive dashboards, performance monitoring, and AI-assisted recommendations.

---

## Problem Statement

District Education Officers (DEOs) and school administrators often rely on static reports to evaluate student performance. This makes it difficult to identify weak regions, monitor learning outcomes, and take timely corrective actions.

LGIS addresses this challenge by providing a centralized analytics platform that highlights learning gaps, identifies at-risk regions, and supports data-driven decision making.

---

## Key Insights Identified

* Math performance is significantly lower than reading performance across several grades.
* Certain states consistently lag behind top-performing regions.
* Reading performance has improved over time, while math improvement remains slower.
* Many students continue to remain at basic proficiency levels despite progressing through higher grades.
